定义opt参数

首先是设置opt基本参数，比较重要的参数如下：

  • –name default=’experiment_name’
  • –checkpoints_dir default=’./checkpoints’

以上参数主要是在visualizer = Visualizer(opt)初始化中确定log和model保存的路径

  • –model = test 使用test_model.py中定义的测试模型
  • –phase = test
  • –epoch = ‘lastest’ 加载最近的epoch保存的模型作为测试模型，

问题及解决

  • 第一次运行中由于save_epoch_freq=5导致存储的中间结果太多硬盘空间不足，清理硬盘后重新输入python train.py --continue_train后继续训练，其代码逻辑在主函数中执行model.setup(opt)调用base_model.py中setup方法，该方法的作用是Load and print networks; create schedulers
  • 报错RuntimeError: Input type (torch.cuda.FloatTensor) and weight type (torch.DoubleTensor) should be the same


    该问题是由于输入input的type是torch.cuda.float64，而模型的参数的type是torch.cuda.float32，解决方案：input = input.float()将其转化为float32即可
